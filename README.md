## 🛠 制作物（ポートフォリオ）

### 1. CSV自動処理ツール (Pythonスクリプト)
- **概要**: 大量のCSVファイルから表記ゆれを自動修正し、システムが受理可能な形式に整えるツールです。
- **想定シーン**: 営業事務時代に経験した「手打ち入力による形式エラー」の解消を想定。全角・半角の混在や不要な空白をプログラム側で自動吸収し、受注ソフトへのインポートミスを未然に防ぎます。
- **こだわり**: 特定の業務に依存せず使い回せるよう、項目名を抽象化（Name, Status等）し、再利用性を高めた設計にしています。
- **使用技術**: Python (標準ライブラリのみ)
- [👉 リポジトリを見る](https://github.com/n-saito-dev/csv-auto-tool)

### 2. CSVデータ管理 & 履歴保存システム (Webアプリ)
- **概要**: ブラウザからCSVをアップロードし、データベース(SQLite)へ保存・閲覧できるアプリです。
- **想定シーン**: チーム内でのExcel共有で発生しがちな「最新版がどれか分からない」「誤操作でデータが消える」といった問題の解決。営業事務員全員が常に同じ最新情報にアクセスできる環境を想定しました。
- **こだわり**: 「いつ・誰が更新したか」の履歴を残すことで、管理業務の透明性と信頼性を向上させています。
- **使用技術**: Python, Flask, SQLAlchemy, SQLite, Pandas, HTML/CSS
- [👉 リポジトリを見る](https://github.com/n-saito-dev/csv_web_app)
