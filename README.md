### 2. CSVデータ管理 & 履歴保存システム (Webアプリ)
- **概要**: ブラウザからCSVをアップロードし、データベース(SQLite)へ保存・閲覧できるアプリです。
- **想定シーン**: チーム内でデータを共有し、過去のアップロード履歴をいつでも振り返れる環境を想定して開発しました。
- **使用技術**: Python, Flask, SQLAlchemy, SQLite, Pandas, HTML/CSS
- [👉 リポジトリを見る](https://github.com/n-saito-dev/csv_web_app)
